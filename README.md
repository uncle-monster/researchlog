# Research Log VS Code 插件

`researchlog` 是一个为科研人员设计的 VS Code 插件，旨在方便地记录在项目不同时间点的科研想法和进展。

## 功能

*   **自动创建日志目录**: 在您的项目根目录下自动创建一个名为 `researchlog` 的文件夹来存放所有的科研日志。
*   **每日日志文件**: 通过简单的命令或快捷键，您可以快速创建当天的日志文件。
*   **命名约定**: 日志文件以数字命名，该数字代表从项目开始记录日志以来的天数（例如，`0.md`, `1.md` 等），让您能够清晰地追溯思路的演变。
*   **Markdown 支持**: 日志文件为 Markdown 格式，方便您利用丰富的格式来组织想法。

## 如何使用

1.  打开命令面板 ( `Ctrl+Shift+P` )。
2.  输入并选择 `Researchlog: 创建当天科研日志`。

或者，您也可以直接使用快捷键 `Ctrl+Alt+R` 来快速创建当天的日志文件。

## 目录结构

插件会在您的项目工作区创建如下的目录结构：

```
.
├── your-project-files/
└── researchlog/
    ├── 0.md
    ├── 1.md
    └── ...
```
